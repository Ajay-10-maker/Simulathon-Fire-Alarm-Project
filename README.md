 Fire Alarm System (Simulathon Project)

 Overview
This project is a Fire Alarm System developed during a **Simulathon (Simulation Hackathon)**. It detects fire hazards using sensors and triggers alerts using buzzer and LED indicators.

 Objective
To design a system that detects fire at an early stage and alerts users instantly to prevent damage and ensure safety.

 Tools & Technologies
- Embedded C
- Keil uVision
- Proteus Simulation
- Microcontroller (8051)

 Features
- Real-time fire detection
- Buzzer alert system
- LED indication
- Simulation-based testing

 Components Used
- Temperature Sensor / Smoke Sensor
- Microcontroller (8051)
- Buzzer
- LED

  Working Principle
The system continuously monitors environmental conditions using sensors. When the detected value exceeds a threshold, the microcontroller activates an alarm (buzzer + LED), indicating a fire hazard.

  Simulation
This project was designed and tested using Proteus simulation.

  Event
Developed during a **Simulathon (Simulation Hackathon)**.
